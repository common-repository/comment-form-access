== Description ==
This plugin gives two methods to limit access to comment form, that one can use alone or combined. The first one limits access to URLs that have a given predefined key access in their query string, the second one to hosts that IP(v4/v6) adresses belongs to a given white list. By default, once this plugin is activated, the comment form doesn't show up anymore except for URLs that use the given random key access. Comment Form Access's form lives in "Settings".

This plugin offers two methods to limit access to the comment form.

== Installation ==
* PHP >= 5.2 needed.
* Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
